Import From Pajek File

This plugin extracts the network structure from a Pajek .net file and displays the data in Constellation. Note that at this stage it does not collect attributes.

Parameters

  • Pajek File - The Pajek File from which to retrieve the graph
  • Retrieve Transactions - Toggles whether the plugin will import nodes only or transactions as well as nodes.

New graph objects are added with the following attributes:

Source Nodes

  • Label

Destination Nodes

  • Label

Transactions

  • Count
